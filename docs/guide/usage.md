## Можливості програми

Програма призначена для виконання базових операцій з числовими даними у консольному режимі.

Підтримувані операції:

**Додавання:**
```python
# add.py
print(5 + 2)

**Віднімання:**
# subtract.py
print(10 - 4)
**Множення:**
# multiply.py
print(3 * 4)
**Ділення:**
# divide.py
print(8 / 2)
